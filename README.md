# cbxwooproductsummary
WooCommerce Product Summary - Shortcode

```
[cbxwooproductsummary id="product id here"]
[cbxwooproductsummary sku="product sku here"]
```


Shortcode Params and Default Values

```
id				 =>
sku				 => ''
show_title       => 1
show_rating      => 1
show_price       => 1
show_excerpt     => 1
show_add_to_cart => 1
show_meta        => 1
show_sharing     => 1
```
